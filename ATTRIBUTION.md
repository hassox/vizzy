# Attribution

## Earth imagery

Bundled under `apps/viz/public/`:

| File | Typical origin |
|---|---|
| `earth-night.jpg` | Night-lights / city-lights composite (commonly NASA Earth Observatory / Black Marble lineage; also distributed with [three-globe](https://github.com/vasturiano/three-globe) examples) |
| `earth-topology.png` | Grayscale topography bump map (same three-globe example lineage) |

These assets are widely redistributed with globe.gl / three-globe demos. If you replace them, keep this file updated and respect the upstream license for any new imagery.

## Libraries

Runtime dependencies are declared in each package’s `package.json` (notably [globe.gl](https://github.com/vasturiano/globe.gl), [three](https://threejs.org/), [ws](https://github.com/websockets/ws)). See those projects for their licenses.
